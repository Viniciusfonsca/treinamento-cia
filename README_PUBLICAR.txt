PUBLICAÇÃO RÁPIDA

1. Acesse netlify.com e faça login.
2. Na tela inicial, arraste a pasta ou o arquivo ZIP deste pacote para a área de deploy.
3. O site será publicado automaticamente.
4. Se quiser usar Meta Pixel, abra o arquivo index.html e troque YOUR_PIXEL_ID pelo ID real.
5. O botão de WhatsApp e o formulário já estão configurados.
